# Tool Assets

Place your tool SVG files in this directory:

- `hand.svg` - Hand (starting tool)
- `wooden.svg` - Wooden Axe/Pickaxe
- `copper.svg` - Copper Axe/Pickaxe  
- `iron.svg` - Iron Axe/Pickaxe
- `golden.svg` - Golden Axe/Pickaxe
- `diamond.svg` - Diamond Axe/Pickaxe
- `netherite.svg` - Netherite Axe/Pickaxe

These will be displayed next to your cursor while mining and will animate with a circular swinging motion!

## Sibling Folder
There should also be an `assets/pickaxe levels/` folder with the same files for displaying in the popup UI.
